# 💳 Caixa Eletrônico em C#

## 📖 Sobre o projeto

Este projeto simula um caixa eletrônico em uma aplicação de console desenvolvida em C#.

O objetivo é praticar lógica de programação utilizando estruturas condicionais, laços de repetição, métodos e manipulação de variáveis.

## ✨ Funcionalidades

- Consultar saldo
- Depositar dinheiro
- Sacar dinheiro
- Encerrar o programa

## 🛠️ Tecnologias utilizadas

- C#
- .NET SDK
- Visual Studio Code

## 🎯 Objetivo

Este projeto foi desenvolvido como parte dos meus estudos em C#, com foco na prática de lógica de programação antes de aprender Orientação a Objetos.
